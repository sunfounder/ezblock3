快速用户指南
=====================================

EzBlock 3.2的主要优化是将在线服务切换至离线模式，并增加了项目文件的导入和导出功能。
届时，新用户注册通道将会关闭，已注册的用户在登录后会收到提示，告知将项目转移到离线模式。
经过一段时间后，在线服务将会关闭。

.. note::
    3.2版本更新后，无需为树莓派重新烧录镜像。
