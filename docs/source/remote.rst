如何使用远程控制功能？
==============================================

1. 进入 **远程控制页面**。

    .. image:: img/img_remote1.png

2. 拖拽您想要的 **小部件**，相应的配置也会弹出。

    .. image:: img/img_remote3.png

3. 返回到 **编程页面**，您会看到新的编程块已经出现。请完成您的代码。

    .. image:: img/img_remote5.png

4. 运行代码后，返回到 **远程控制页面**，开始使用它。

    .. image:: img/img_remote6.png
