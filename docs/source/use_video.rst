ビデオ機能の使用方法は？
====================================

1. **Start** に **camera monitor on** のブロックをドラッグします。

    .. image:: img/img_remote_camera1.png

2. **Remote Control Page** で、 **Video** ウィジェットをドラッグし、プロジェクトを実行します。

    .. image:: img/img_remote_camera2.png

3. これでビデオが表示されます。

    .. image:: img/img_remote_camera4.png
